Youth Colony Data Collection & Analytics Dashboard

During the summer of 2024, I contributed to a youth colony organized by multiple churches, aiming to streamline participant registration and centralize all related data. The project involved building a complete data pipeline, from structured collection to interactive visualization, while ensuring privacy and accuracy.

The process began with a KoboToolbox form to collect registrations over a two-week period. Once the registration period ended, the data was exported in CSV format. I then used Python scripts to clean the dataset, handle missing values, remove duplicates, and standardize encoding and formatting, ensuring consistent, reliable data across all participating churches.

After preprocessing, the data was imported into Google Sheets for final organization and collaborative access. Finally, the cleaned data was visualized using Looker Studio, producing an interactive dashboard that provided insights on age distribution, gender balance, participation by church, and attendance by activity. This dashboard was delivered to church coordinators to support planning and decision-making.

For portfolio purposes, all sensitive information was replaced with a synthetic dataset, preserving structure and analytical value without compromising privacy. This project demonstrates hands-on experience in data collection, cleaning, transformation, and visualization, applied to a real-world community initiative.

Key Features

Structured Data Collection

A custom KoboToolbox form ensured clean, validated, and uniform data input across multiple churches.

Automated Data Cleaning

Python scripts were used to preprocess, standardize, and anonymize the dataset while preserving analytical value.

Interactive Visual Dashboard

LockerStudio dashboards provided clear insights: age groups, gender ratios, participation by church, and activity attendance.

Privacy-Preserving Synthetic Dataset

All original data was replaced with generated values to maintain confidentiality without impacting analysis quality.
